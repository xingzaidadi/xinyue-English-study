# Weekly Planning Rules

## Default Week Shape

| Day | Focus |
|---|---|
| Monday | Reading + vocabulary |
| Tuesday | Listening + error review |
| Wednesday | Writing + sentence upgrade |
| Thursday | Reading + speaking |
| Friday | Listening + vocabulary |
| Saturday | Practice set or diagnostic task |
| Sunday | Light input, no formal testing |

## Time Budget

- Weekdays: 35-50 minutes.
- Weekend: 45-90 minutes only if the learner is calm and school workload allows it.
- One no-test day per week.

## Planning Output

For each day include:

- date
- status
- estimated time
- tasks
- completion notes block

Use checkboxes for tasks.

## Avoid

- Full mock tests before foundational readiness.
- More than one heavy skill per weekday.
- Long correction sessions after a difficult task.
- Changing the route based on one bad day.
