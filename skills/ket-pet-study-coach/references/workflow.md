# Workflow Reference

## Daily Start

1. Pull latest repository changes.
2. Read `state/current-status.md`.
3. Read `state/current-week.md`.
4. Read current month log.
5. Give today's tasks in Chinese.

## Daily Finish

1. Parse completion feedback.
2. Update `state/current-week.md`.
3. Update `logs/YYYY-MM.md`.
4. Adjust tomorrow only if needed.
5. Commit and push.

## Weekly Review

Review:

- completion rate
- reading progress
- listening progress
- writing output
- speaking confidence
- emotional load
- repeated error causes

Then update `state/current-week.md`.
